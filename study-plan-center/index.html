🎓 Galala University Study Plan Center

Database-Driven Study Plan Management System

Welcome to the New Study Plan Center

This is the new database-driven system for managing Galala University study plans. The system is currently under development and will be integrated with the main website.

✅ System Status: The new React-based Study Plan Center has been successfully built and is ready for integration. All components are functional and tested.

Key Features

👥 Student Portal

Browse faculties, programs, years, semesters, and courses with complete prerequisite information.

⚙️ Admin Dashboard

Manage faculties, programs, and courses without coding. Changes appear instantly for students.

⚡ Real-time Sync

Automatic data refresh every 2 seconds. Students see admin changes immediately.

🎨 Galala Branding

Complete Galala University color scheme and design throughout the system.

📱 Responsive Design

Optimized for mobile, tablet, and desktop devices.

🔐 Secure Access

Role-based access control with admin authentication.

📋 Implementation Details

Technology Stack:

  • Frontend: React 19 + Tailwind CSS 4 + shadcn/ui
  • Backend: Express 4 + tRPC 11
  • Database: MySQL + Drizzle ORM
  • Authentication: Manus OAuth

Database Schema:

  • Faculties - Academic faculties
  • Programs - Degree programs
  • Years - Academic years
  • Semesters - Semesters within years
  • Courses - Individual courses
  • Course Prerequisites - Prerequisite relationships
